# Log Sentinel — Akıllı Log Analiz & Güvenlik Motoru
Acik kaynak kodlu ve Discord entegrasyonlu threat detection sistemi.
Sunucu loglarınızı (Nginx, Apache vb.) gerçek zamanlı okuyup SQLi, XSS ve Brute Force gibi siber tehditleri otomatik tespit eden defansif güvenlik aracı. Saldırganın IP ve lokasyon (GeoIP) analizini yaparak tespit ettiği tehditleri anlık olarak Discord kanalınıza bildirir. Hızlı, hafif ve tamamen açık kaynaklı.
